# 🛠️ MAoS — Manager Assist of Support

> Десктопное приложение для специалистов технической поддержки розничной сети kari.com.  
> Версия **3.7.5** · Python + PyQt5 · Windows

---

## 📋 Описание

MAoS (Manager Assist of Support) — внутренний инструмент службы поддержки. Объединяет в одном окне всё необходимое для диагностики, управления и документирования инцидентов в магазинах сети: карточку магазина, сетевые инструменты, управление роутерами, базу знаний, SQL-запросы и оформление отчётов с помощью ИИ.

---

## 🚀 Возможности

### 🏪 Карточка магазина
- Загрузка данных магазина по номеру через REST API (ADM)
- Отображение IP-адресов (WAN / LAN), реквизитов, контактов
- Автоподстановка IP в другие вкладки приложения

### 📡 Пинг
- Пинг магазина по номеру
- Параллельная проверка нескольких адресов
- Горячая клавиша: `Ctrl+E`

### 🔭 Сканер сети
- Сканирование диапазона IP-адресов
- Определение активных хостов
- Горячая клавиша: `Ctrl+T`

### 🖥️ ПК (RMS)
- Удалённое подключение к рабочим станциям магазинов
- Горячая клавиша: `Ctrl+R`

### 📡 Роутеры
Автоматическое определение типа роутера по адресу `rt{номер_магазина}`:

| Тип | Протокол определения |
|-----|----------------------|
| **Hongdian** | `/gui/login.cgi` в URL |
| **MikroTik** | `RouterOS` в теле ответа |
| **ASUS (Tomato)** | Ответ на порту 8080 |

Для каждого типа отображаются: внешний IP, внутренний IP, прошивка, аптайм и другие данные.

**Генератор конфигов (Hongdian):**
- Кнопка «⚙️ Конфиги» открывает диалог создания `.conf`-файла
- Автоподстановка IP WAN и LAN из данных роутера
- Поля сгруппированы в два блока: **WAN** (IP, шлюз, маска, DNS) и **VPN** (логин, пароль)
- Файл сохраняется с именем `{номер_магазина}.conf`

### 📋 Отбивки (база знаний)
- Хранилище инструкций и решений в виде дерева папок
- Поиск по названию и содержимому
- Поддержка HTML-форматирования, изображений, PDF
- Drag & drop между категориями
- Экспорт / импорт через ZIP
- **Импорт из TXT-файлов** — добавление одного или нескольких файлов сразу с выбором категории
- Сохранение отдельной отбивки в PDF
- Горячая клавиша: `Ctrl+G`

### 🗄️ SQL
- Выполнение SQL-запросов к базам данных магазинов
- Горячая клавиша: `Ctrl+S`

### 📷 Видеорегистраторы (DVR)
- Быстрый доступ к видеорегистраторам магазинов

### 🤖 ИИ — Оформление отчётов
Заполнение шаблона инцидента с помощью AI (OpenRouter / GPT):

1. Вставить **текст заявки** от пользователя
2. Описать **что было сделано** для решения
3. Нажать «⚡ Оформить отчёт»

Результат оформляется по шаблону:
```
[ПРОБЛЕМА]
[НЕ ХВАТАЛО В ЗАЯВКЕ]
[ДИАГНОСТИКА]
[ПРИЧИНА]
[РЕШЕНИЕ]
```

### 📞 Линия PBX
- Мониторинг очереди агентов Asterisk (pbx.kari.local)
- Прозрачный оверлей поверх всех окон со списком активных агентов
- Скрытие/показ оверлея: `Ctrl+O`
- Настройка угла экрана и прозрачности

### 🎮 Развлечения
- Встроенная игра в **Дурак** для нескольких игроков через сетевую шару

| Клавиши | Действие |
|---------|----------|
| `Ctrl+E` | Пинг магазина |
| `Ctrl+R` | Подключение к ПК (RMS) |
| `Ctrl+T` | Сканирование сети |
| `Ctrl+G` | Отбивки |
| `Ctrl+S` | SQL-запросы |
| `Ctrl+O` | Скрыть/показать оверлей |

## 🔄 Автообновление

Приложение автоматически проверяет наличие новых версий при запуске через GitHub Releases:

- **Обычное обновление** — заменяет только `MAoS.exe`
- **Полное обновление** — скачивает и распаковывает `MAoS_full.zip` (PowerShell)
- Репозиторий релизов: `daniokpavlov18-collab/MAoS-releases`
- При разрыве соединения — автоматически 3 попытки повтора

## ⚙️ Настройки

Доступны через иконку шестерёнки:

- **Тема** — тёмная / светлая с анимированным переключателем
- **Авторизация** — логин/пароль для ADM и учётные данные роутеров (Hongdian, ASUS, MikroTik)
- **Пути** — WinBox, пути к сетевым ресурсам
- **Оверлей** — монитор, угол размещения, прозрачность

## 🏗️ Технологии

| Компонент | Стек |
|-----------|------|
| UI | PyQt5 |
| Сборка | PyInstaller `--onedir` |
| Роутеры | HTTP REST (Hongdian), RouterOS API port 8778 (MikroTik), Basic Auth (ASUS) |
| Карточка магазина | REST API (ADM) |
| ИИ | OpenRouter API (`openai/gpt-oss-120b:free`) |
| Дурак (мультиплеер) | JSON-файлы на сетевой шаре `\\RU-EKT-B1-PWS96\MAoS_Records` |
| Обновления | GitHub Releases + urllib |
| База знаний | JSON (локальный файл) |
